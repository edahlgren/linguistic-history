Check preprocessing and d3.js loading

This demo transforms the volume1-poster data into a JSON graph, loads it into the browser using d3.js, and visualizes it using a vertical arc diagram as a verification step.

While interesting, the vertical arc diagram visualization isn't necessarily important. It was used here as a test, to get familiar with the structure of the data, to show how d3.js can be used to load and visualize the data, and to check whether people or links are missing.
